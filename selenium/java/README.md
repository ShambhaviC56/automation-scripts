## Setup

### Prerequisites
- Java 11 or higher
- Maven 3.6+
- Chrome, Firefox, or Edge browser

### Environment Variables
Set the following environment variables before running tests:

```bash
# Required
export SALESFORCE_LOGIN_URL=https://your-instance.salesforce.com
export SALESFORCE_USERNAME=your-username
export SALESFORCE_PASSWORD=your-password

# Optional
export BROWSER=chrome        # chrome, firefox, edge
export HEADLESS=false       # true for headless mode
export DEFAULT_TIMEOUT=30   # timeout in seconds
```

### Install Dependencies
```bash
mvn clean install -DskipTests
```

## Running Tests

### Run all tests
```bash
mvn clean test
```

### Run specific test class
```bash
mvn clean test -Dtest=TC001Test
```

### Run with specific browser
```bash
mvn clean test -Dbrowser=firefox
```

### Run in headless mode
```bash
mvn clean test -Dheadless=true
```

### Run with Maven profile
```bash
mvn clean test -Pheadless
```

## Page Object Model Pattern

### Locators (Object Repository)
Store all element locators in dedicated classes:

```java
package locators;

import org.openqa.selenium.By;

public class AccountLocators {
    public static final By NEW_BUTTON = By.cssSelector("button[name='New']");
    public static final By ACCOUNT_NAME_INPUT = By.cssSelector("lightning-input-field[field-name='Name'] input");
    public static final By SAVE_BUTTON = By.cssSelector("button[name='SaveEdit']");
}
```

### Page Objects
Create page classes with methods for user actions:

```java
package pages;

import org.openqa.selenium.WebDriver;
import locators.AccountLocators;

public class AccountPage extends BasePage {
    
    public AccountPage(WebDriver driver) {
        super(driver);
    }
    
    public void createAccount(String accountName) {
        click(AccountLocators.NEW_BUTTON);
        fill(AccountLocators.ACCOUNT_NAME_INPUT, accountName);
        click(AccountLocators.SAVE_BUTTON);
        waitForSpinner();
    }
}
```

### Test Classes
Write tests using page objects:

```java
package tests.account;

import org.testng.Assert;
import org.testng.annotations.Test;
import pages.AccountPage;
import tests.BaseTest;

public class TC001Test extends BaseTest {
    
    @Test
    public void testCreateAccount() {
        AccountPage accountPage = new AccountPage(driver);
        accountPage.createAccount("Test Account");
        Assert.assertTrue(accountPage.isSuccessToastDisplayed());
    }
}
```

## Key Features

- **Automatic Login/Logout**: BaseTest handles authentication
- **Thread-safe WebDriver**: DriverFactory uses ThreadLocal for parallel execution
- **Configurable Timeouts**: Set via environment variables or TestConfig
- **Cross-browser Support**: Chrome, Firefox, Edge
- **Headless Mode**: Run without browser UI
- **Salesforce-specific**: Built for Lightning Experience
